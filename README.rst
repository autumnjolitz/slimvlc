Requirements
---------------

    - VLC (obtainable via ``brew install --cask vlc``)
    - Python 3.12 (obtainable via ``brew install python@3.12``)


Installation
----------------


Development::

    python -m pip install -e .


Production::

    python -m pip install git+https://github.com/autumnjolitz/slimvlc.git#egg=slimvlc


Running
------------

    python3 -m slimvlc --help
